# Roadmap de Produto Smartzap

## Criação de Conteúdo

### Pesquisas de Opinião (Novo)
#### Coleta de dados Qualitativos e Quantitativos

##### V1.1 - Criação (Admin)
**Pesquisa Qualitativa**
- [ ] Perguntas Abertas (Open-Ended)
- [ ] Sem limite de criação
- [ ] Configurar Nome e Pergunta
- [ ] Sem impacto na performance
- [ ] [Testar Criação](app://create-course)

**Pesquisa Quantitativa**
- [ ] Múltipla Escolha (Single Select)
- [ ] Configurar Nome, Pergunta e Alternativas
- [ ] Até 5 alternativas
- [ ] Sem alternativa "correta"
- [ ] Sem impacto na performance

**Pesquisa NPS**
- [ ] Escala 1 a 5
- [ ] Escala 1 a 10
- [ ] Configurar Pergunta

##### V1.1 - Consumo (Aluno)
**Experiência de Resposta**
- [ ] Link de redirecionamento
- [ ] Texto Aberto (máx 2000 caracteres)
- [ ] Seleção única sem feedback (Quantitativo)
- [ ] Bloqueio: Obrigatório responder para finalizar
- [ ] Botão "Voltar para WhatsApp"

**Fluxo NPS**
- [ ] Pesquisa NPS 1-5
- [ ] Pesquisa NPS 1-10
